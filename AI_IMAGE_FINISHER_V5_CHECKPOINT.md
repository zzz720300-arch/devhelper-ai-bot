PR #20 — `AI Image Finisher V5: four real engines Android Engine Gate`

## Зафиксированная цель Engine Gate

До разработки финального пользовательского интерфейса должны реально выполниться внутри Android:

1. Aire 0.18.1 — профессиональное изменение размера.
2. BiRefNet Lite через ONNX Runtime — основной качественный движок удаления фона.
3. U2NetP через ONNX Runtime — только быстрый резервный режим.
4. ncnn/Real-ESRGAN — отдельная модель x2 и отдельная модель x4; x2 запрещено получать уменьшением результата x4.
5. VTracer 0.6.4 через Rust/JNI — настоящий редактируемый SVG с элементами `<path>` и без встроенного растра.

## Зафиксированная архитектура

- отдельные модули движков;
- единый integration API;
- технический APK только для Engine Self-Test;
- ABI: `arm64-v8a` и `x86_64`;
- Android SDK 36;
- Build Tools 36.1.0;
- JDK 17;
- Gradle 8.13;
- никакого финального UI до прохождения Engine Gate;
- никакого открытия внешнего приложения для выполнения обработки;
- модели и native-библиотеки считаются подключёнными только после фактического вызова на Android.

## Зафиксированная проверка

CI обязан:

1. сгенерировать полный Android-проект из обычных текстовых bootstrap-файлов;
2. скачать и проверить SHA-256 моделей и native-зависимостей;
3. собрать ncnn JNI и VTracer JNI для двух ABI;
4. собрать отдельные AAR;
5. собрать технический APK;
6. установить APK в Android-эмулятор;
7. выполнить self-test всех четырёх функций;
8. выдать артефакты только при полном успехе.
